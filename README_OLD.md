## 📖 使用方式

### 1. 瀏覽目錄

```bash
cat CATALOG.md
```

### 2. 查看分類

```bash
ls -lh books/
```

### 3. 閱讀書籍

```bash
# 推理懸疑類
cat "books/推理懸疑/無人生還_阿加莎·克里斯蒂_TXT小说天堂.md"

# 盜墓探險類
cat "books/盜墓探險/鬼吹灯之云南虫谷_天下霸唱.md"
```

### 4. 搜索內容

```bash
# 搜索書名
grep -r "關鍵詞" books/

# 搜索作者
grep -l "阿加莎" books/推理懸疑/*.md
```

## 🗑️ 質量控制

### 過濾標準

- ✅ 最小文件大小: 10 KB
- ✅ 最小內容長度: 5000 字符
- ✅ 最少章節數: 3 章(短篇除外)
- ✅ 亂碼檢測: < 5% 異常字符
- ✅ 去重: 內容哈希去重

### 被移除的內容

查看 `DELETION_REPORT.md` 了解:

- 被移除的書籍列表
- 移除原因
- 原始文件位置

所有廢料文件已移動到 `廢料/` 目錄,保持原始目錄結構。

## 📞 維護

### 重新整理

```bash

```

### 添加新書

1. 將 Markdown 文件放入對應分類目錄
2. 更新 `metadata.json`
3. 重新生成目錄: `python3 generate_catalog.py`
