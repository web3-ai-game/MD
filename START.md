# 🚀 快速啟動指南

## 啟動閱讀器

```bash
cd /mnt/sms/MD
npm start
```

服務器將運行在: **http://localhost:3000**

## 功能說明

### ✅ 已實現
- **分類瀏覽**: 6個分類,1016本書籍
- **搜索功能**: 實時搜索書名
- **閱讀器**: Markdown渲染,舒適閱讀體驗
- **進度保存**: 自動保存閱讀位置(每30秒)
- **書籤功能**: 添加/管理書籤
- **響應式設計**: 適配不同屏幕

### 🎯 使用方式

1. **瀏覽書籍**
   - 左側點擊分類展開書籍列表
   - 點擊書名開始閱讀

2. **搜索書籍**
   - 頂部搜索框輸入關鍵詞
   - 實時顯示匹配結果

3. **保存進度**
   - 點擊右上角"💾 保存進度"
   - 或等待自動保存(30秒)

4. **書籤管理**
   - 點擊"📑 書籤"打開面板
   - 查看和跳轉到書籤位置

## 故障排除

### 端口被占用
```bash
# 查找占用端口的進程
lsof -i :3000

# 或使用其他端口
PORT=8080 npm start
```

### 數據庫鎖定
```bash
# 刪除數據庫重新初始化
rm reading.db
npm start
```

### 依賴問題
```bash
# 重新安裝
rm -rf node_modules package-lock.json
npm install
```
